## Instalação

### 1. Clonar o repositório

```bash
git clone https://github.com/thiagop90/hotel-management.git
```

### 2. Instalar dependências

```bash
npm install
```

### 3. Execute o aplicativo

```bash
npm run dev
```

O aplicativo estará disponível em `http://localhost:3000`.
